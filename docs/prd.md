# Product Requirements Document (PRD)

هذا الملف هو مدخل الـ PRD وفق إعدادات `.bmad-core/core-config.yaml`.

- المستندات المفصلة موجودة ضمن `docs/prd/` بما فيها `PRD.md`
- استخدم هذا الملف كنقطة دخول ملخّصة وروابط إلى الأقسام الفرعية

روابط:
- المستند الرئيسي: `docs/prd/PRD.md`
- تجربة المستخدم: `docs/prd/USER-EXPERIENCE.md`
- تصميم واجهة المستخدم: `docs/prd/UI-UX-DESIGN.md`
- المعمارية: `docs/prd/ARCHITECTURE.md`
