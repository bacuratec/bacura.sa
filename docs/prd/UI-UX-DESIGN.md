# وثيقة تصميم واجهة المستخدم وتجربة المستخدم (UI/UX Design)
## منصة باكورة أعمل - Bakora Amal Platform

---

## 1. نظرة عامة على التصميم

### 1.1 فلسفة التصميم
تصميم بسيط وحديث يركز على سهولة الاستخدام والوضوح، مع دعم كامل للغة العربية واتجاه RTL.

### 1.2 المبادئ التصميمية
- **البساطة:** واجهة نظيفة بدون تعقيد
- **الوضوح:** معلومات واضحة وسهلة الفهم
- **التناسق:** تصميم موحد في جميع الصفحات
- **الاستجابة:** يعمل على جميع الأجهزة
- **إمكانية الوصول:** قابل للوصول لجميع المستخدمين

## 2. نظام التصميم (Design System)

### 2.1 الألوان (Color Palette)

#### 2.1.1 الألوان الأساسية
```
Primary Color:     # (يتم تحديده حسب هوية العلامة التجارية)
Secondary Color:   # 
Success Color:     #10B981 (أخضر)
Error Color:       #EF4444 (أحمر)
Warning Color:     #F59E0B (برتقالي)
Info Color:        #3B82F6 (أزرق)
```

#### 2.1.2 الألوان المحايدة
```
Background:        #FFFFFF (أبيض)
Surface:           #F9FAFB (رمادي فاتح)
Text Primary:      #111827 (أسود/رمادي داكن)
Text Secondary:    #6B7280 (رمادي)
Border:            #E5E7EB (رمادي فاتح)
```

#### 2.1.3 استخدام الألوان حسب الدور
- **المدير:** ألوان إدارية احترافية
- **مقدم الخدمة:** ألوان تعكس الخدمة والجودة
- **طالب الخدمة:** ألوان ودودة وسهلة الاستخدام

### 2.2 الطباعة (Typography)

#### 2.2.1 الخطوط
- **العربية:** خط عربي واضح (Cairo, Tajawal, أو خط مخصص)
- **الإنجليزية:** Sans-serif (Inter, Roboto, أو خط مخصص)

#### 2.2.2 أحجام الخطوط
```
Heading 1 (H1):    2.5rem (40px) - العناوين الرئيسية
Heading 2 (H2):    2rem (32px) - العناوين الثانوية
Heading 3 (H3):    1.5rem (24px) - العناوين الفرعية
Body Large:        1.125rem (18px) - النصوص الكبيرة
Body:              1rem (16px) - النصوص العادية
Body Small:        0.875rem (14px) - النصوص الصغيرة
Caption:           0.75rem (12px) - التسميات التوضيحية
```

### 2.3 المسافات (Spacing)
```
xs:   0.25rem (4px)
sm:   0.5rem (8px)
md:   1rem (16px)
lg:   1.5rem (24px)
xl:   2rem (32px)
2xl:  3rem (48px)
3xl:  4rem (64px)
```

### 2.4 الحدود والظلال (Borders & Shadows)
```
Border Radius:
  sm:  0.25rem (4px)
  md:  0.5rem (8px)
  lg:  0.75rem (12px)
  xl:  1rem (16px)
  full: 9999px (دائري)

Shadows:
  sm:  0 1px 2px rgba(0,0,0,0.05)
  md:  0 4px 6px rgba(0,0,0,0.1)
  lg:  0 10px 15px rgba(0,0,0,0.1)
  xl:  0 20px 25px rgba(0,0,0,0.1)
```

## 3. المكونات الأساسية (Core Components)

### 3.1 الأزرار (Buttons)

#### 3.1.1 أنواع الأزرار
- **Primary Button:** الإجراءات الرئيسية
- **Secondary Button:** الإجراءات الثانوية
- **Outline Button:** أزرار بحدود
- **Ghost Button:** أزرار شفافة
- **Danger Button:** إجراءات خطيرة (حذف، إلغاء)

#### 3.1.2 أحجام الأزرار
- **Small:** للأزرار الصغيرة
- **Medium:** الحجم الافتراضي
- **Large:** للأزرار الكبيرة

### 3.2 حقول الإدخال (Input Fields)

#### 3.2.1 أنواع الحقول
- **Text Input:** النصوص
- **Email Input:** البريد الإلكتروني
- **Password Input:** كلمة المرور
- **Phone Input:** رقم الهاتف
- **Select Dropdown:** القوائم المنسدلة
- **Textarea:** النصوص الطويلة
- **File Upload:** رفع الملفات
- **Date Picker:** اختيار التاريخ

#### 3.2.2 حالات الحقول
- **Default:** الحالة الافتراضية
- **Focus:** عند التركيز
- **Error:** عند وجود خطأ
- **Disabled:** معطل
- **Success:** عند النجاح

### 3.3 البطاقات (Cards)
- **Project Card:** بطاقة المشروع
- **Request Card:** بطاقة الطلب
- **User Card:** بطاقة المستخدم
- **Service Card:** بطاقة الخدمة
- **Statistics Card:** بطاقة الإحصائيات

### 3.4 الجداول (Tables)
- **Data Table:** جدول البيانات مع البحث والتصفية
- **Sortable Columns:** أعمدة قابلة للترتيب
- **Pagination:** ترقيم الصفحات
- **Row Actions:** إجراءات الصف

### 3.5 النماذج (Modals)
- **Confirmation Modal:** تأكيد الإجراءات
- **Form Modal:** نماذج في نافذة منبثقة
- **Info Modal:** عرض المعلومات
- **Delete Modal:** تأكيد الحذف

### 3.6 التنقل (Navigation)

#### 3.6.1 القائمة الجانبية (Sidebar)
- **Admin Sidebar:** قائمة المدير
- **Provider Sidebar:** قائمة مقدم الخدمة
- **Icons:** أيقونات واضحة لكل قسم
- **Active State:** تمييز الصفحة النشطة

#### 3.6.2 القائمة العلوية (Header)
- **Logo:** شعار المنصة
- **Navigation Links:** روابط التنقل
- **User Menu:** قائمة المستخدم
- **Notifications:** الإشعارات
- **Language Switcher:** تبديل اللغة

### 3.7 الحالات (States)

#### 3.7.1 حالات التحميل
- **Loading Spinner:** دوار التحميل
- **Skeleton Screen:** هيكل الصفحة أثناء التحميل
- **Progress Bar:** شريط التقدم

#### 3.7.2 حالات الخطأ
- **Error Message:** رسالة الخطأ
- **Empty State:** حالة فارغة
- **404 Page:** صفحة غير موجودة

#### 3.7.3 حالات النجاح
- **Success Toast:** إشعار النجاح
- **Success Icon:** أيقونة النجاح

## 6. التصميم المتجاوب (Responsive Design)

### 6.1 نقاط التوقف (Breakpoints)
```
Mobile:      < 640px
Tablet:      640px - 1024px
Desktop:     > 1024px
Large:       > 1280px
```

### 6.2 استراتيجية التصميم المتجاوب
- **Mobile First:** التصميم يبدأ من الموبايل
- **Flexible Grid:** شبكة مرنة
- **Responsive Images:** صور متجاوبة
- **Touch-Friendly:** أزرار كبيرة للمس

### 6.3 التكيف حسب الجهاز

#### 6.3.1 الموبايل
- قائمة منبثقة (Hamburger Menu)
- بطاقات عمودية
- أزرار كاملة العرض
- إخفاء العناصر غير الضرورية

#### 6.3.2 التابلت
- قائمة جانبية قابلة للطي
- بطاقات في شبكة 2 أعمدة
- أزرار متوسطة الحجم

#### 6.3.3 سطح المكتب
- قائمة جانبية ثابتة
- بطاقات في شبكة 3-4 أعمدة
- جميع العناصر مرئية

## 7. إمكانية الوصول (Accessibility)

### 7.1 معايير WCAG
- **Contrast Ratio:** نسبة تباين كافية (4.5:1 للنصوص)
- **Keyboard Navigation:** التنقل بالكيبورد
- **Screen Reader Support:** دعم قارئ الشاشة
- **Focus Indicators:** مؤشرات التركيز واضحة

### 7.2 الممارسات
- استخدام `alt` للصور
- استخدام `aria-label` للأزرار
- استخدام العناوين بشكل صحيح (H1, H2, ...)
- استخدام الألوان مع النصوص وليس فقط

## 8. الرسوم المتحركة (Animations)

### 8.1 أنواع الرسوم المتحركة
- **Page Transitions:** انتقالات الصفحات
- **Hover Effects:** تأثيرات التحويم
- **Loading Animations:** رسوم التحميل
- **Toast Notifications:** إشعارات منبثقة
- **Modal Animations:** رسوم النوافذ المنبثقة

### 8.2 المبادئ
- **Subtle:** خفيفة وغير مزعجة
- **Purposeful:** لها هدف واضح
- **Fast:** سريعة (أقل من 300ms)
- **Smooth:** سلسة

## 9. دعم RTL (Right-to-Left Support)

### 9.1 التكيف مع العربية
- **Direction:** `dir="rtl"`
- **Text Alignment:** محاذاة النصوص لليمين
- **Icons:** عكس الأيقونات عند الحاجة
- **Layout:** عكس التخطيط

### 9.2 المكتبات المستخدمة
- `tailwindcss-rtl` - دعم RTL في Tailwind
- `tailwindcss-dir` - دعم الاتجاه

- توجيهات واضحة
