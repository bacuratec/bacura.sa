# Epic 4: Provider Rates

## Description
عرض وإدارة أسعار مقدم الخدمة مع تنسيقات عملة وواجهة سهلة.

## Acceptance Criteria
1. عرض الأسعار بعملة SAR بالتنسيق المحلي الصحيح
2. دعم التصفية والترتيب حسب الخدمة/السعر
3. تحسين الأداء بقوائم كبيرة باستخدام افتراضية عند الحاجة
4. رسائل تنبيه مترجمة عند نجاح/فشل التحديثات
