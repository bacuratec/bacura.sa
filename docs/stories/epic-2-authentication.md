# Epic 2: Authentication

## Description
تحسين تسجيل الدخول/الخروج وتدويل رسائل الأخطاء وتجربة المستخدم أثناء الإرسال.

## Acceptance Criteria
1. عرض رسائل أخطاء مترجمة عند فشل تسجيل الدخول
2. تعطيل زر الإرسال أثناء العملية ومنع النقرات المكررة
3. استرجاع الملف الشخصي بعد تسجيل الدخول بنجاح
4. تسجيل الخروج يعمل ويعيد الحالة إلى عدم تسجيل الدخول
