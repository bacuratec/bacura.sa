# Epic 3: Requests Exploration

## Description
تحسين صفحة استكشاف الطلبات بقوائم، فلاتر، وترقيم صفحات قابل للتخصيص.

## Acceptance Criteria
1. إضافة فلاتر حالة/مدينة/تصنيف تعمل مع الـ API
2. ترقيم الصفحات فعّال ويحافظ على الحالة عبر التنقل
3. مؤشرات تحميل واضحة بدون تجميد الواجهة
4. عدم فقدان الفلاتر عند العودة من تفاصيل الطلب
