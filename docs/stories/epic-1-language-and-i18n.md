# Epic 1: Language & i18n

## Description
تحسين نظام اللغة والاتجاه لضمان تجربة سلسة بين العربية والإنجليزية دون إعادة تحميل أو مشاكل توافق.

## Acceptance Criteria
1. تبديل اللغة يتم فورًا ويحدّث النصوص والاتجاه
2. لا توجد تحذيرات hydration أو وميض بصري أثناء التبديل
3. حفظ اللغة في `localStorage` ويعاد تطبيقها عند التحميل
4. التوافق مع أصناف Tailwind `rtl:` و`ltr:` في الواجهات الحساسة
