# Story 7.1: Notification Center UI

## Status
Completed

## Story
**As a** user,
**I want** to see my notifications in a dropdown menu,
**so that** I can stay updated on my requests and orders.

## Acceptance Criteria
1. Add a notification bell icon in the header.
2. Show a dropdown with the list of notifications (read/unread).
3. Clicking a notification marks it as read and navigates to the relevant page.
4. "Mark all as read" button.
