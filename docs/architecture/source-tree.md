# Source Tree Overview

- `app/` Next.js app router pages and layouts
- `src/components/` UI components (layouts, shared, feature modules)
- `src/context/` React contexts (LanguageContext)
- `src/lib/` i18n init, redux store/providers, supabase client
- `src/locales/` `ar.json`, `en.json` translations
- `src/utils/` helpers (env, routing, image, validation)
- `src/views/` page-level view composition by domain
- `.bmad-core/` BMAD agents, tasks, templates, workflows
- `docs/` PRD and architecture docs; stories at `docs/stories/`
